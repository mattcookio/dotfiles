# Alacritty Configuration

## Features

- Gruvbox color scheme
- JetBrains Mono Nerd Font
- True color support
- Cursor customization
- Scrollback buffer
- Padding and window size

## Key Bindings

- `Ctrl+Shift+` - Increase font size
- `Ctrl+-` - Decrease font size
- `Ctrl+0` - Reset font size
- `Ctrl+Shift+C` - Copy
- `Ctrl+Shift+V` - Paste
